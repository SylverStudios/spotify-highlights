# How to use the UI package

```sh
yarn # default command is install
yarn test
yarn start # localhost:3000
# Share with friends!
ngrok http 3000 -host-header="localhost:3000"
```

## Production

Build production assets (js and css together) with:

```sh
npm run prod
```
